Lapitar is a new open source Minecraft avatar service, providing 2D and 3D avatars with a simple API.

Faces

A simple, flat avatar with only the face of the player's skin.

https://tar.lapis.blue/face/[size]/<player>[.png]

Example:

https://tar.lapis.blue/face/128/Aesen_.png

Heads

A perspective avatar with the head of the player's skin.

https://tar.lapis.blue/avatar/[size]/<player>[.png]

Example:

https://tar.lapis.blue/avatar/128/Aesen_.png

Skins

The raw skin of the player. This will return the player's skin as-is, with no modifications.

https://tar.lapis.blue/skin/<player>[.png]

Example:

https://tar.lapis.blue/skin/Aesen_.png
